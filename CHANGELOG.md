# 更新日志

本项目版本遵循 [SemVer](https://semver.org/lang/zh-CN/)。

## [0.2.1] - 2026-06-12

### 补全更新时间

- 新增 `scripts/enrich_codeberg.py`：调 Codeberg(Gitea) API 补全托管在
  codeberg.org 的项目的最后更新时间/Star/归档状态。上游 awesome-selfhosted
  只抓 GitHub，导致这些项目「更新时间」显示未知。
- 接入每日 CI（`fetch_data → enrich_codeberg → build_dataset`）。
- 效果：「更新时间」未知项 200 → 162，补全 38 个（多为活跃项目）。

## [0.2.0] - 2026-06-12

### 重做分类体系（小白友好）

- **两级分类**：把原 83 个技术标签（含「杂项」）归并成 16 个一看就懂的大类，
  每个大类下带中文小类。点大类展开/收起小类，复选小类即可筛选。
  - 大类：媒体与影音、通讯与社交、笔记与文档、建站与博客、任务与日程、
    财务与商务、文件与备份、阅读与学习、服务器与网络、数据与监控、
    自动化与物联网、安全与隐私、AI 与开发、游戏、生活与健康、实用工具。
  - 原「杂项」73 个软件已逐个归类，**不再有杂项**。
- **新增「更新时间」筛选**：按项目最后更新分档（🟢近1年 / 🟡1-2年 / 🟠2-3年 /
  🔴3年以上 / ⚪未知），方便筛掉很久没更新的老项目。
- **删除「语言/平台」「许可证」筛选区**：普通用户用不到，侧栏更清爽。
- 分类映射集中在 `scripts/categories.py`，CI 每日重建数据时自动套用。
